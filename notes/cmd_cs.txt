
Start Container with display connection:
docker run --name r2_humble -e DISPLAY=host.docker.internal:0.0 -it osrf/ros:humble-desktop-full


Write command to bashrc:
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc


Create a new terminal for existing container:
docker exec -it b05ab4130a19 bash


Build docker file from folder where Dockerfile exists:
docker build -t new_image_name .
